**Linux vs Windows**

On a system with:

- Ryzen 7 5700G CPU
- 64 GB RAM 3600 MHz CL18
- Nvidia RTX 4070 Ti GPU

The authors observe a 58% speedup on Linux compared to Windows. The performance sweetspot is reached with 4 game instances on Linux, and 2 on Windows.
